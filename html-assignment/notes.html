Things I have learned

    The paragraph tag
    < p > < / p > is the tag. This will open the code for the ability to write a paragraph on the webpage.
    The bold tag
    < b > < / b > is to open and close the code for making a word or sentence bold.
    The navigation tag
    < nav > < / nav > would open a tag to where one could navigate and local other links.
    The list tag
    < li > < / li >. This tag is used for creating lists of items.
    The blockquote
    < blockquote > < / blockquote > is the tag used to create a quote. Can be used for whatever. Also if it is cited you can use the cite tag. < cite > < / cite >
