Things I have learned

    The paragraph tag
    The bold tag
    This is to open and close the code for making a word or sentence bold.
    The navigation tag
    The navigation tag would open a tag to where one could navigate and local other links.
    The list tag
    This tag is used for creating lists of items.
    The blockquote
    Blckquote is the tag used to create a quote. Can be used for whatever. Also if it is cited you can use the cite tag.
